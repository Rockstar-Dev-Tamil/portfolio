## 🌟 Overview
Welcome to my personal portfolio repository! I'm a B.Tech CSE student, Developer, and Problem Solver focusing on C, C++, Python, web development, and game scripting. This portfolio acts as a central hub for all my projects, showcasing both my coding abilities and eye for premium UI design.

## ✨ Key Features
- **Cinematic Intro**: A custom, glitch-styled `INITIALIZING SYSTEMS...` splash screen on load.
- **Glassmorphism UI**: Beautiful frosted-glass headers, cards, and subtle transparent borders.
- **3D Interactive Cards**: Powered by [Vanilla-Tilt.js](https://micku7zu.github.io/vanilla-tilt.js/), making all project cards tilt interactively on hover with dynamic glare.
- **Typing Animations**: Integrates [Typed.js](https://mattboldt.com/demos/typed-js/) to dynamically cycle through developer roles in the hero section.
- **Responsive Layout**: Adapts perfectly to mobile and desktop screens using modern CSS properties.
- **Atmospheric Backgrounds**: Floating background orbs and slowly panning radial gradients give the site a distinct "cyber" atmosphere.
- **Performance Optimized**: Zero heavy frameworks—just pure, polished Vanilla HTML, CSS, and JS.

## 📥 Local Development
To run this portfolio locally, you simply need to serve the `index.html` file.
1. Clone the repository:
   ```bash
   git clone https://github.com/Rockstar-Dev-Tamil/portfolio.git
   ```
2. Navigate to the directory:
   ```bash
   cd portfolio
   ```
3. Open `index.html` in your favorite browser, or use an extension like VS Code Live Server.
